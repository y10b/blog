## 🎯 주제 전략 (1000개 구성)

### A. 실전 사례 시리즈 (400개, 40%)
**특징**: 실제 경험 기반, 숫자 포함, 신뢰도 높음

**예시**:
- "디자이너→PM 전환 3년, 연봉은 어떻게 변했나"
- "피그마 플러그인으로 월 300만원 부수입 과정"
- "블로그 애드센스 2주 승인 완벽 가이드"
- "YouTube 구독자 1천→3천명 3개월 전략"
- "Claude Code로 MVP 2주 만든 과정"

### B. 심화 가이드 (300개, 30%)
**특징**: 기존 주제 3배 확장, 단계별 상세 가이드

**예시**:
- PM A/B 테스트 → 3편 시리즈
  1. "통계적 유의성 계산 완벽 가이드"
  2. "A/B 테스트 실패 사례 10가지"
  3. "Notion A/B 테스트 추적 시스템"

### C. 도구 완전 정복 (200개, 20%)
**특징**: 각 도구별 심화 시리즈

**주요 도구**:
- Linear (20편)
- Figma (30편)
- Notion (20편)
- Cursor (15편)
- Claude Code (15편)
- v0.dev (10편)

### D. 문제 해결 Q&A (100개, 10%)
**특징**: 실제 독자 질문 기반

**예시**:
- "PM 면접 자주 나오는 질문 50개"
- "디자이너-개발자 협업 문제 30가지"
- "리모트 PM의 커뮤니케이션 문제 해결"

## 🛠 배치 생성 시스템 아키텍처

### 1. 주제 생성 파이프라인
```
scripts/
├── generate-1000-topics.ts      # 1000개 주제 리스트 생성
├── generate-weekly-batch.ts     # 주간 5개 배치 생성
└── validate-quality.ts          # 품질 검증 자동화
```

### 2. 주간 워크플로우

#### 월요일: 키워드 조사 & 선정
```bash
# 다음 주 5개 주제 선정
pnpm tsx scripts/select-weekly-topics.ts
```

#### 화-수요일: AI 생성
```bash
# 5개 포스트 AI 생성 (DRAFT 상태)
pnpm tsx scripts/generate-weekly-batch.ts
```

#### 목-금요일: 편집 & 검토
- 실제 사례 추가
- 이미지/다이어그램 삽입
- 내부 링크 연결
- 품질 체크리스트 확인

#### 토요일: 예약 발행
- 최종 검토
- 2-3일 간격 예약 발행 설정
